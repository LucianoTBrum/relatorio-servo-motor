1. Introdução

Este Relatório descreve o projeto de controle de um servo motor utilização de um potenciometro em um arduino, com o objetivo de desenvolver habilidades em eletronica e programação.

2. Objetivos

O principal objetivo é fazer o controle da direção do servo motor com a utilização do potenciometro em um microcontrolador Arduino.

3. Materiais utilizados

  • Arduino
  • 1 Potenciometros
  • 1 Servo motor
  • Protoboard
  • Fios de conexão

4. Diagrama do circuito

Abaixo está o diagrama do circuito projetado no Tinkercad:

5. Código Fonte

            #include 
                Servo myservo;
                int potpin = 0;
                int val;
                
                void setup() {
                  myservo.attach(9);
                }           
                
                void loop() {
                  val = analogRead(potpin);
                  val = map(val, 0, 1023, 0, 180);
                  myservo.write(val);
                  delay(15);
                }

6. Procedimentos

O potenciometro e o servo motor foram conectados aos pinos digitais e analogicos do Arduino conforme o diagrama, e o código foi carregado na placa.

7. Resultados

O braço do servo motor se moveu conforme o esperado, alternando sua direção conforme os valores fossem alterados com a utilização do potenciometro.

8. Discussão

O funcionamento do circuito aconteceu de forma satisfatoria e conforme esperado

9. Conclusão

O projeto foi bem sucedido, permitindo a pratica dos conceitos de eletronica e programação.
